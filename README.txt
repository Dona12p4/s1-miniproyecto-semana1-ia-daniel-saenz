S1_Miniproyecto_Semana1_materiales

Este directorio contiene la entrega del miniproyecto de Semana 1 sobre viabilidad inicial de una tarea de IA con datos de una celda industrial.

Organizacion de archivos

- data/
  Contiene el dataset usado en el notebook:
  datos_celda_industrial_sinteticos.csv

- notebooks/
  Contiene el notebook principal de la entrega:
  S1_Miniproyecto_Semana1_Informe_viabilidad_IA.ipynb

- docs/
  Contiene la bitacora de uso de IA generativa:
  bitacora_ia.md

- src/
  Carpeta reservada para codigo auxiliar si el proyecto se amplia.

Reproducibilidad

Para ejecutar el notebook en otro computador, conserve la carpeta completa con la misma estructura. El notebook carga el dataset usando rutas relativas, por lo que puede estar en Escritorio, Descargas u otra ubicacion, siempre que el archivo CSV siga dentro de data/.

Formas recomendadas de ejecucion:

1. Abrir la carpeta S1_Miniproyecto_Semana1_materiales en Jupyter, VS Code o el entorno usado por el docente.
2. Abrir notebooks/S1_Miniproyecto_Semana1_Informe_viabilidad_IA.ipynb.
3. Ejecutar las celdas de arriba hacia abajo.

GitHub o evidencia de organizacion

Si el docente solicita GitHub, se debe incluir el enlace al repositorio donde este esta misma estructura de archivos. Si no se solicita GitHub, esta carpeta organizada funciona como evidencia de entrega, porque incluye el dataset, el notebook y la bitacora de uso de IA.
